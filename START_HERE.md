## ğŸ”§ Customization (Super Easy!)

### Change Animation Speed
Open any component file and modify:
```tsx
transition={{ duration: 0.6 }}  // Change this number
```
- 0.3 = Very fast
- 0.6 = Normal
- 1.0 = Slow

### Change Colors
Look for Tailwind classes like:
```tsx
className="from-blue-600 to-cyan-600"  // Change these
```

### Disable Mouse Trail
In `src/App.tsx`, comment out:
```tsx
// <MouseTrail />
```

### Adjust 3D Scene
Edit `src/components/ThreeBackground.tsx`:
- Particle count (line 30)
- Sphere properties (lines 48-60)
- Star count (lines 62-80)
